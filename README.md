# OurBlog

## 前言
基于springboot与vue开发的多人博客平台，小学期作业

## 使用方法

将仓库克隆至本地

```bash
git clone git@github.com:bluehyssopu/OurBlog.git
```

安装依赖

```bash
npm install
```

启动项目

```bash
npm run serve
```
